DOS基本命令

发表于2022-08-25|更新于2022-08-27|DOS

Dos命令

[百度百科](DOS命令_百度百科 (baidu.com))

Windows系统不区分大小写

退出终端: exit

清除位置盘符: @echo off

输出字符: echo 字符名 注意: % 变量名%

清屏: cls

等待用户操作: pause

切换盘符: 盘符名 : 或 cd / 盘符名 :

访问当前目录下所有文件: dir

切换同一级目录: cd 文件名

返回上一级目录: cd …

创建目录: md 目录名

删除目录: rd目录名

目录拷贝 : copy

创建文件: cd > 文件名 . 后缀名

删除文件: del 文件名

文件剪切 : move

算术运算: set/a 1+2

运算符: + - * / %(取余)

重定向: > >> < <<

​ **箭头方向表示文件输入方向 **

​ 单箭头表示覆盖,双箭头表示追加

多命令运算: && ||

​ && : 只有 正确 -> ; 错误 X->

​ || : 执行正确的

管道操作运算: |

查找: find “条件”

netstat: 这是一个用来查看网络状态的命令，操作简便功能强大。

​ -an查看本地机器的所有端口

​ -a 查看本地机器的所有开放端口，可以有效发现和预防木马，可以知道机器所开的服务等信息。

​ 这里可以看出本地机器开放有FTP服务、Telnet服务、邮件服务、WEB服务等。用法：netstat -a IP。

​ -r 列出当前的路由信息，告诉我们本地机器的网关、子网掩码等信息。用法：netstat -r IP。

​ (444端口连接可能中木马了)

批处理命令基本格式: 命令 子命令 参数 操作 /? /help 获取帮助文档

参数的传递 : % num %1 %2 ~~~

注释 : rem 注释内容

cmd窗口设置 : color title

时间命令 : date time

启动命令: start

调用其它bat文件: call bat文件名

任务列表查看 : tasklist

终止任务进程 : taskkill

文件夹结构命令 : tree

关机命令 : shutdown

计划任务命令 : schtasks

文件夹重命名命令 : ren

用户操作命令 : net user

用户组操作命令 : net localgroup

主机联通性检测命令 : ping

网络路由信息命令 : tracert

网络适配器命令 : ipconfig

ARP信息命令 : arp

条件判断结构 : if-else

判断文件是否存在 : exists

循环遍历文件夹名称 : for

交互操作跳转语句 : goto

代码块 : : 名称
